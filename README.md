# 3dsketchinsolidworks
Proyek teknis 2D ke dalam model parametrik 3D yang akurat menggunakan SolidWorks. Melalui proyek ini, dilakukan penerapan berbagai teknik pemodelan untuk memastikan bahwa komponen tidak hanya benar secara visual, tetapi juga memenuhi standar fungsionalitas manufaktur, terutama dalam aspek aliran fluida dan kekuatan struktur pada titik sambungan.
