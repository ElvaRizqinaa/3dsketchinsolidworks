# 3dsketchinsolidworks
Proyek ini bertujuan untuk mendemonstrasikan dokumentasi teknis 2D ke dalam model parametrik 3D yang akurat menggunakan SolidWorks. Fokus utama dari proyek ini adalah sebuah komponen dalam sistem industri yang memerlukan presisi tinggi pada bagian internal bore dan socket.

Melalui proyek ini, dilakukan penerapan berbagai teknik pemodelan untuk memastikan bahwa komponen tidak hanya benar secara visual, tetapi juga memenuhi standar fungsionalitas manufaktur, terutama dalam aspek aliran fluida dan kekuatan struktur pada titik sambungan.


## **Spesifikasi Teknis & Tools yang Digunakan:**
* **Extruded Boss/Base**: Digunakan untuk membentuk geometri bodi utama pipa vertikal dan pipa cabang horizontal.
* **Extruded Cut (Through All)**: Digunakan untuk melubangi pipa vertikal agar mendapatkan lubang tembus sempurna untuk aliran fluida.
* **Socket Dimensioning**: Penerapan lubang bertingkat pada ujung pipa dengan diameter untuk kebutuhan fitting.
* **Advanced Filleting (Radius)**: Diterapkan pada pertemuan bodi pipa untuk memperkuat struktur sambungan serta untuk aspek keamanan dan estetika.


###  Metodologi Desain
Proses pengerjaan dilakukan dengan urutan langkah kerja sistematis:
1. **Analisis Blueprint**: Menerjemahkan simbol diameter, radius (R), dan pandangan potongan (*Section View*).
2. **Sketching & Constraint**: Membuat sketsa bersifat *Fully Defined* (garis hitam) untuk memastikan akurasi dimensi.
3. **Feature Management**: Penggunaan fitur *Mid Plane* pada ekstrusi untuk menjaga simetri model.
4. **Verifikasi Internal**: Pengecekan rongga dalam menggunakan *Section View* untuk memastikan semua sudah terhubung dengan benar.
